# lazyuv Roadmap

v1 covers the project workflow (dependencies, sync/lock, scripts). Planned next:

## Milestone 2 — Environments
- Python version management (`uv python install`, `uv python pin`).
- Virtualenv management (`uv venv`).

## Milestone 3 — Full dashboard
- Global tools / uvx (`uv tool install/list/upgrade/uninstall`).
- Cache management (`uv cache dir/clean/prune`).

## Backlog
- Project browser/switcher (open a project other than the CWD).
- Dependency tree view (`uv tree`).
- Outdated/upgrade view.
- Build & publish (`uv build`, `uv publish`).
